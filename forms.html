Vocab definitions (Forms)

Regular Bottom Defines a submit bottom which submits all forms values to a form-handler.
"for" attribute The form attribute has a value which is the ID of the form element it relates to.
Attribute Provides additional information about elements.
HTML inputs HTML inputs are field or controls in a web form used to collect data such as text, number, data, etc. various input typer allow for different kind of entry.
Checkboxes Checkbox is a square box that is ticked (checked) when activated.
Fieldset Fieldset is an HTML element used to group related elements within a form.
Legend Legend is used to provide a caption for the fieldset element.
Form Method (post and get) Post is used to send data to a server to create/update resources unsecure.
Form Action The action attribute specifies where to send the form data when a form is submitted.
