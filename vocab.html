Vocab definitions
Layer a layer communication parts of the software
Authorization Function os specifying rights priviliger for neccesary resources, allowing access for someone to get data or information.
User Interface the means by which the user and a computer system interact, in particular the use of input devices and software.
Application Software A program designed to handles specific task for user.
Logic Layer Is the part of program where classes determine the information needed to solve problems, and manipulate the in formation and return results to the presentation layer for formating.
Authentication Verifying the identify of a user process or device often a prerequisite to allowing access to resources in an information system.
Error-Handling Is the process of responding and recovering from error conditions in your program.
Persistence Layer Is a software that is similar to database and saves/gets data from a storage.
Error-messages Is an error message is the information displayed when an unforeseen problem occurs, usually on a computer or some other device.
Logging Is the act of keeping a log of event that occurs in a computer system, such a problem errors or just information on current operation.
Cross-cutting concerns Cross-cutting concerns are parts of a program that rely on or must affect many other parts of the system
